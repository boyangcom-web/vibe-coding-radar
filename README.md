# Vibe Coding雷达

📡 **每周帮你找到值得做、做得出来、做完有反馈的 AI 项目**

刚装好 Codex、Claude Code 等 AI 编码工具，不知道该做什么项目？
Vibe Coding雷达 从 **最好玩、最好用、最好搓** 三个方向，为你精选适合新手的开源项目灵感。

## 三大分类

| 分类 | 说明 |
|------|------|
| 🎮 **最好玩** | 有探索感、能激发创意的项目 |
| 🔧 **最好用** | 解决实际问题，能长期使用的工具 |
| 🛠 **最好搓** | 硬件、实体、小设备，适合动手党 |

## 功能

- **项目雷达** — 三个方向浏览项目库
- **本周明星项目** — 自动追踪 GitHub Trending，推荐适合新手复刻的开源项目
- **开工提示词** — 每个项目附带可直接复制给 AI 编码助手的提示词
- **自动更新** — 每周一自动同步最新热门项目到 GitHub

## 技术栈

- 原生 JavaScript + CSS + HTML
- 纯前端，无需构建工具
- PowerShell 自动化脚本

## 每周自动更新

通过 Windows 任务计划程序，每周一 10:00 自动运行：

1. 更新本周 GitHub 明星项目数据
2. 同步到 GitHub 仓库

## 本地运行

直接打开 `首页.html` 即可浏览。
