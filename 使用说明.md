# Vibe Coding雷达 — 小白使用指南

## 这是什么？

一个帮你找 AI 编程项目的网页工具。刚装好 Codex、Claude Code 这类 AI 编程助手，不知道做什么？打开这个页面，它会从三个方向给你推荐适合新手练手的开源项目。

每个项目都附带了**可以直接复制给 AI 的开工提示词**——复制粘贴就能让 AI 帮你写代码。

---

## 怎么打开？

### 方法一：直接双击（最快）

找到 `首页.html` 文件，双击，浏览器会自动打开。

> 如果双击后是代码界面而不是网页，右键 → 打开方式 → 选择 Chrome / Edge / Firefox。

### 方法二：命令行启动（更稳定，推荐）

1. 在文件夹地址栏输入 `cmd` 回车，打开命令行窗口
2. 输入下面这行，回车：
   ```
   python3 -m http.server 8080
   ```
3. 打开浏览器，地址栏输入 `http://localhost:8080/首页.html`
4. 看到页面后，就可以用了
5. 用完在命令行窗口按 `Ctrl + C` 关闭

## 页面怎么用？

整个页面分为五大区域，从上往下分别是：

### 1️⃣ 顶部导航栏

页面最上方，固定在屏幕顶部。有五个按钮：

| 按钮 | 作用 |
|------|------|
| **Vibe Coding雷达** | 点一下回到顶部 |
| **项目雷达** | 跳到项目列表 |
| **开工提示词** | 跳到提示词区 |
| **明星项目** | 跳到本周热门 |
| **新手阶段** | 跳到新手分类 |

右上角还有三个工具：

- **中 / EN** — 点击切换中英文界面
- **复制自动更新提示词** — 复制一段指令给 AI，让它帮你自动更新项目数据
- **GitHub** — 跳转到项目源码

### 2️⃣ 首屏区域

打开页面第一眼看到的大区域。

- 中间有一个**旋转雷达图**，三个标签「最好玩」「最好用」「最好搓」可以点
- 右侧卡片显示当前精选项目，点「查看项目详情」会跳到下面项目区
- 雷达图下方的筛选条可以过滤方向

### 3️⃣ 新手阶段（三个方向）

用三张卡片解释项目的三个方向：

| 方向 | 适合谁 | 做什么 |
|------|--------|--------|
| 🎮 **最好玩** | 想做出有趣东西的人 | 创意交互、AI 生成类项目 |
| 🔧 **最好用** | 想解决实际问题的人 | 效率工具、自动化项目 |
| 🛠 **最好搓** | 喜欢硬件动手的人 | ESP32、传感器、小设备 |

### 4️⃣ 项目雷达（核心功能区）

这里是**最重要**的部分，分左右两栏：

**左栏：项目列表**

- 列出当前分类下的所有项目
- 点任意一个项目，右侧会显示详细信息
- 顶部有「最好玩 / 最好用 / 最好搓」三个标签切换
- 可以按「阶段」（入门友好 / 熟悉基础 / 想进阶）和「周期」（1-2天 / 2-3天 / 3-5天）筛选
- 点「重置」清空筛选条件

**右栏：项目详情**

点左侧项目后，这里出现三个标签页：

- **为什么值得做** — 三条理由，告诉你这个项目好在哪
- **适合阶段** — 说明适合什么水平的人
- **开工提示词** — ⭐ **这是最有用的！** 一段可以直接复制给 AI 的指令

> 👉 **怎么用开工提示词**：点「复制提示词」按钮，然后粘贴到 Codex / Claude Code / ChatGPT 的对话框里，AI 就会按这个需求帮你写代码。

每个项目还附带了原始 GitHub 链接，可以去看原版项目。

### 5️⃣ 本周明星项目

最下方的区域，展示 GitHub 上最近增长最快的开源项目：

- 左侧表格列出项目名、描述、分类
- 顶部可以按「本周增长 / 适合复刻 / AI 工具 / 开发工具」筛选
- 右侧「本周雷达读数」是对热门趋势的解读
- 底部的行动条可以直接复制本周推荐提示词

## PowerShell 自动化脚本（Windows 用户）

项目自带了三个 PowerShell 脚本，可以设置每周自动更新：

### 首次设置.ps1

**做什么**：关联你的 GitHub 账号

**怎么用**：
1. 右键 `首次设置.ps1` → 使用 PowerShell 运行
2. 按提示输入你的 GitHub Personal Access Token（去 github.com → Settings → Developer settings → Personal access tokens 创建一个）
3. 输入你的 GitHub 用户名
4. 脚本会自动创建远程仓库并推送代码

### 安装自动任务.ps1

**做什么**：注册一个 Windows 计划任务，每周一上午 10:00 自动更新项目数据

**怎么用**：
1. **右键 → 使用 PowerShell 运行（以管理员身份）**
2. 脚本会自动创建定时任务
3. 以后每周一 10:00 电脑会自动拉取最新热门项目

### 自动更新.ps1

**做什么**：手动执行一次自动更新（检查数据、提交变更、推送到 GitHub）

**怎么用**：右键运行即可

### 自动更新提示词.md

**做什么**：如果你用的是 AI 编码助手（Codex、Claude Code 等），把这个文件内容复制给 Agent，它会自动帮你完成周更。

## 常见问题

**Q: 打开页面是空白的？**
A: 确保 `样式.css` 和 `交互.js` 和 `首页.html` 在同一个文件夹里。

**Q: 双击 HTML 打开的页面很奇怪？**
A: 用方法二，命令行启动服务器再访问，兼容性最好。

**Q: 图片（素材文件夹里的 PNG）是干什么的？**
A: 概念设计参考图，不影响页面功能。删掉也没关系。

**Q: 不会用 PowerShell？**
A: 不用也没关系。三个脚本是可选的，页面本身双击就能用。脚本只是给想自动更新的人准备的。

**Q: 怎么自己加新项目？**
A: 用文本编辑器（记事本、VS Code 等）打开 `交互.js`，找到 `项目数据` 那一段，照着格式加就行。改完保存刷新页面即可生效。
